🥒 Pickle Party

8 players · 2 courts · nobody sits out

Who's playing today?

Type in all 8 names to get the party started. Everyone plays every game — 4v4 across two courts — so with ~15 min games you'll fit about 8 games each in 2 hours. Winners work their way up to the top court, so matchups stay close and fun. The very last game is always the big one: best vs. best, no matter who's already faced who.

Everyone uses the same code. Whoever adds the players taps Start new party; everyone else taps Join existing party with that same code to see live updates.
